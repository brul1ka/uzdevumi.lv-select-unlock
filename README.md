# Uzdevumi.lv Text Selection Unlock

Userscript that restores normal text selection on uzdevumi.lv.
Some pages disable text selection via frontend JavaScript.  
This script prevents that behaviour locally in the browser.

**It does not modify server data.**
It only changes how the page behaves on the client side.
